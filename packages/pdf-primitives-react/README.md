# @dfactory/pdf-primitives-react

Framework-native React PDF primitives for DFactory templates.

## Highlights

- Typed components for layout, pagination, running content, TOC, tokens, and tables
- Marker semantics mapped to DFactory PDF feature plugins
- Theme variable support via `Document` `theme` prop

## Development

```bash
pnpm --filter @dfactory/pdf-primitives-react typecheck
pnpm --filter @dfactory/pdf-primitives-react build
```
