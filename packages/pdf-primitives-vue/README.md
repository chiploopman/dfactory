# @dfactory/pdf-primitives-vue

Framework-native Vue PDF primitives for DFactory templates.

## Highlights

- Typed Vue components with the same names/semantics as React primitives
- Pagination and running-content primitives mapped to DFactory PDF features
- Theme variable support via `Document` `theme` prop

## Development

```bash
pnpm --filter @dfactory/pdf-primitives-vue typecheck
pnpm --filter @dfactory/pdf-primitives-vue build
```
