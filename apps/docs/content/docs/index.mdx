Welcome to the official dFactory documentation.

dFactory is a Storybook-style platform for PDF template cataloging, previewing, and generation across React and Vue template ecosystems.

## What You Can Do

- Build and preview templates in HTML and PDF modes.
- Author reusable templates with `@dfactory/template-kit`.
- Extend rendering behavior using PDF feature plugins.
- Run production API + UI with one deployable service.
- Operate in monorepo mode with contributor guardrails.
