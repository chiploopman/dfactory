# DFactory Skills Index

Use these repo-local skills when working in this codebase:

1. `shadcn`
   - Official shadcn skill installed from `shadcn/ui`.
   - Trigger for all work in `packages/ui` and follow `packages/ui/components.json`.

2. `dfactory-template-authoring`
   - Trigger when adding or editing template contracts under `src/templates/*/template.tsx`.

3. `dfactory-adapter-development`
   - Trigger when adding framework adapters or modifying adapter contracts.

4. `dfactory-playwright-e2e`
   - Trigger when editing E2E tests, preview flows, and UI/API integration behavior.

5. `dfactory-deployment-targets`
   - Trigger when editing Docker, Kubernetes, Helm, AKS, or EKS artifacts.
